# Security Policy

## Supported Versions

The following versions of our projects are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| >= 2.x  | :white_check_mark: |
| < 2.x   | :x:                |

## Reporting a Vulnerability

Please report any security vulnerabilities to the maintainers of the Strata Game Library.
We will acknowledge your report within 48 hours and provide a fix or mitigation as soon as possible.
